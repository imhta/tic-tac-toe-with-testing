# tic tac toe with testing
Ruby CLI implementation of tic tac toe game with testing 

>Tic-tac-toe (American English), noughts and crosses (British English), or Xs and Os is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. [know more](https://en.wikipedia.org/wiki/Tic-tac-toe)

## Installation
#### Requirement
* Ruby 2.5.0+
* Rspec
#### Run
```
bundle install
ruby bin/main.rb
```
#### Test
```
rspec
```
